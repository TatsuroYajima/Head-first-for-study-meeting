## How to use
## Reveal.js インストール
```sh
npm install -g reveal-md
```

## スライド作成
ルートディレクトリで下記コマンドを実行する。
```sh
reveal-md md -w
```

`-w` オプションによって、リロードなしで変更が反映されるようになる。

## 作成したスライドをHTMLとして出力
```sh
reveal-md md --static docs
```
